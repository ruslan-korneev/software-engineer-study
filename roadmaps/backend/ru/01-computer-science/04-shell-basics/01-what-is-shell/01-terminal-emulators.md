## Что такое эмулятор терминала?

**Эмулятор терминала** — это программа, которая предоставляет текстовый интерфейс для взаимодействия с операционной системой через командную оболочку (shell). Исторически терминалы были физическими устройствами, но сегодня мы используем программные эмуляторы.

## Связь между терминалом и shell

```
Пользователь → Эмулятор терминала → Shell (bash/zsh) → Операционная система
```

- **Эмулятор терминала** — это окно, в котором вы видите текст и вводите команды
- **Shell** — это интерпретатор команд, который обрабатывает ваш ввод

## Популярные эмуляторы терминала

### Linux
- **GNOME Terminal** — стандартный терминал для GNOME
- **Konsole** — терминал для KDE
- **xterm** — классический X11 терминал
- **Terminator** — терминал с поддержкой множества панелей
- **Alacritty** — быстрый терминал на GPU

### macOS
- **Terminal.app** — встроенный терминал
- **iTerm2** — расширенный терминал с множеством функций
- **Alacritty** — кроссплатформенный GPU-ускоренный терминал

### Windows
- **Windows Terminal** — современный терминал от Microsoft
- **PowerShell** — встроенная оболочка
- **Git Bash** — эмулятор bash для Windows
- **WSL Terminal** — терминал для Windows Subsystem for Linux

## Основные функции эмуляторов

1. **Отображение текста** — вывод результатов команд
2. **Ввод команд** — приём пользовательского ввода
3. **Поддержка цветов** — цветовое выделение текста
4. **Прокрутка истории** — просмотр предыдущего вывода
5. **Копирование/вставка** — работа с буфером обмена
6. **Вкладки и панели** — множество сессий в одном окне

## Пример запуска терминала

При открытии терминала вы увидите **приглашение** (prompt):

```bash
username@hostname:~$
```

- `username` — имя пользователя
- `hostname` — имя компьютера
- `~` — текущая директория (домашняя папка)
- `$` — обычный пользователь (`#` — root)

## Настройка терминала

Большинство терминалов позволяют настраивать:
- Шрифт и его размер
- Цветовую схему
- Прозрачность фона
- Горячие клавиши
- Поведение курсора
