## Структура команды в shell

Команда в shell имеет следующую структуру:

```
команда [опции] [аргументы]
```

- **Команда** — имя программы для выполнения
- **Опции** — модификаторы поведения (начинаются с `-` или `--`)
- **Аргументы** — данные, над которыми работает команда

## Примеры структуры

```bash
$ ls                    # только команда
$ ls -l                 # команда + опция
$ ls -l /home           # команда + опция + аргумент
$ ls -l -a /home        # команда + несколько опций + аргумент
$ ls -la /home          # короткие опции можно объединять
```

## Типы опций

### Короткие опции (один дефис)
```bash
$ ls -l                 # один символ после дефиса
$ ls -la                # можно объединять: -l + -a
$ ls -l -a              # или раздельно
```

### Длинные опции (два дефиса)
```bash
$ ls --all              # полное название опции
$ ls --human-readable   # описательные имена
$ ls -l --all           # можно комбинировать с короткими
```

## Базовые команды

### echo — вывод текста
```bash
$ echo "Hello, World!"          # вывести текст
$ echo $HOME                    # вывести значение переменной
$ echo -n "No newline"          # без перевода строки в конце
$ echo -e "Line1\nLine2"        # интерпретировать escape-последовательности
```

### date — дата и время
```bash
$ date                          # текущая дата и время
$ date +%Y-%m-%d                # форматированная дата: 2024-12-26
$ date +%H:%M:%S                # только время: 14:30:45
$ date -d "next friday"         # дата следующей пятницы (Linux)
```

### cal — календарь
```bash
$ cal                           # текущий месяц
$ cal 12 2024                   # декабрь 2024
$ cal -3                        # три месяца: пред., текущий, след.
$ cal 2025                      # весь 2025 год
```

### whoami и id — информация о пользователе
```bash
$ whoami                        # имя текущего пользователя
$ id                            # подробная информация (uid, gid, группы)
$ id username                   # информация о другом пользователе
```

### uname — информация о системе
```bash
$ uname                         # название ядра (Linux, Darwin)
$ uname -a                      # вся информация
$ uname -r                      # версия ядра
$ uname -m                      # архитектура (x86_64, arm64)
```

### uptime — время работы системы
```bash
$ uptime
14:30:45 up 5 days, 3:22, 2 users, load average: 0.15, 0.10, 0.05
```

### hostname — имя компьютера
```bash
$ hostname                      # имя хоста
$ hostname -I                   # IP-адреса (Linux)
```

## Работа с дисками и памятью

### df — дисковое пространство
```bash
$ df                            # использование дисков
$ df -h                         # в человекочитаемом формате
$ df -h /home                   # для конкретной директории
```

### free — использование памяти (Linux)
```bash
$ free                          # использование RAM
$ free -h                       # человекочитаемый формат
$ free -m                       # в мегабайтах
```

### du — размер директорий
```bash
$ du                            # размер текущей директории
$ du -h                         # человекочитаемый формат
$ du -sh *                      # размер каждого элемента в текущей папке
$ du -sh /var/log               # размер конкретной папки
```

## Команды для текста

### head и tail — начало и конец файла
```bash
$ head file.txt                 # первые 10 строк
$ head -n 5 file.txt            # первые 5 строк
$ tail file.txt                 # последние 10 строк
$ tail -n 20 file.txt           # последние 20 строк
$ tail -f /var/log/syslog       # следить за файлом в реальном времени
```

### wc — подсчёт строк, слов, символов
```bash
$ wc file.txt                   # строки, слова, байты
$ wc -l file.txt                # только строки
$ wc -w file.txt                # только слова
$ wc -c file.txt                # только байты
```

## Полезные практики

1. **Используйте `--help`** для справки по любой команде:
   ```bash
   $ ls --help
   ```

2. **Не бойтесь экспериментировать** — большинство команд безопасны

3. **Комбинируйте опции** для нужного результата:
   ```bash
   $ ls -lah     # подробно, включая скрытые, человекочитаемо
   ```
