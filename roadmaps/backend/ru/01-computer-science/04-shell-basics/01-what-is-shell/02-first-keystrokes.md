## Начало работы в терминале

Когда вы открываете терминал, вы видите **приглашение командной строки** (prompt). Это означает, что shell готов принимать команды.

## Ввод команд

Просто введите команду и нажмите **Enter**:

```bash
$ date
Thu Dec 26 12:00:00 MSK 2024
```

Команда `date` показывает текущую дату и время.

## Базовые команды для начала

### Дата и время
```bash
$ date          # текущая дата и время
$ cal           # календарь текущего месяца
$ cal 2024      # календарь на весь год
$ cal -3        # календарь: прошлый, текущий и следующий месяц
```

### Информация о системе
```bash
$ whoami        # имя текущего пользователя
$ hostname      # имя компьютера
$ uname -a      # информация о системе
```

### Работа с дисковым пространством
```bash
$ df            # свободное место на дисках
$ df -h         # в человекочитаемом формате (GB, MB)
$ free          # информация о памяти (Linux)
```

## Важные клавиши

### Enter
Выполняет введённую команду

### Ctrl + C
**Прерывает** выполнение текущей команды. Если команда "зависла" — нажмите Ctrl+C.

```bash
$ sleep 100     # команда будет ждать 100 секунд
^C              # нажали Ctrl+C — команда прервана
```

### Ctrl + D
**Выход** из терминала (или завершение ввода). Это то же самое, что команда `exit`.

### Ctrl + L
**Очищает** экран терминала. Эквивалент команды `clear`.

### Tab
**Автодополнение** — одна из самых полезных клавиш. Начните вводить команду или путь и нажмите Tab.

```bash
$ dat<Tab>      # автоматически дополнится до date
$ cd /ho<Tab>   # автоматически дополнится до /home/
```

## Несколько команд в одной строке

```bash
$ date; cal     # выполнить обе команды последовательно
$ date && cal   # выполнить cal только если date успешна
```

## Перенос длинных команд

Если команда не помещается в одну строку, используйте `\`:

```bash
$ echo "Это очень длинная строка, \
которая продолжается на следующей строке"
```

## Советы для новичков

1. **Регистр важен** — `Date` и `date` — разные команды
2. **Пробелы важны** — `cd/home` не работает, нужен пробел: `cd /home`
3. **Не бойтесь экспериментировать** — большинство команд безопасны
4. **Ctrl+C — ваш друг** — если что-то пошло не так, прервите выполнение
