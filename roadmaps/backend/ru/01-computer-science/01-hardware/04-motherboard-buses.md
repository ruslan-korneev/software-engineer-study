## Материнская плата

Главная плата, соединяющая все компоненты: CPU, RAM, накопители, периферию.

### Ключевые элементы
- **Сокет CPU** — разъём для процессора
- **Слоты RAM** — для оперативной памяти
- **Чипсет** — управляет потоками данных
- **PCIe слоты** — для видеокарт, NVMe, сетевых карт
- **SATA/M.2** — для накопителей
- **BIOS/UEFI** — прошивка для загрузки

## Шины (Buses)

Проводники для передачи данных между компонентами.

### PCIe (основная шина)

| Версия | x1 | x4 | x16 |
|--------|----|----|-----|
| PCIe 3.0 | 1 GB/s | 4 GB/s | 16 GB/s |
| PCIe 4.0 | 2 GB/s | 8 GB/s | 32 GB/s |
| PCIe 5.0 | 4 GB/s | 16 GB/s | 64 GB/s |

### Другие интерфейсы

| Шина | Скорость | Применение |
|------|----------|------------|
| SATA III | 600 MB/s | HDD, бюджетные SSD |
| USB 3.2 | 20 Gb/s | Периферия |
| Thunderbolt 4 | 40 Gb/s | Внешние устройства |

## BIOS vs UEFI

| BIOS | UEFI |
|------|------|
| Старый | Современный |
| Диски до 2 TB | Диски любого размера |
| Текстовый | Графический интерфейс |

## Для Backend-разработчика

- **IPMI/BMC** — удалённое управление сервером
- **10 Gb/s+ сеть** — для высоких нагрузок
- **DMA** — диск ↔ RAM без участия CPU
