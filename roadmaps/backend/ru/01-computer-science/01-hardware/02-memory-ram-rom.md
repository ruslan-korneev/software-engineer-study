## Иерархия памяти

```
Скорость ↑    Объём ↓    Цена ↑
┌───────────┐
│ Регистры  │  ~1 нс     Байты
├───────────┤
│   Кэш     │  1-20 нс   KB-MB
├───────────┤
│   RAM     │  ~100 нс   GB
├───────────┤
│ SSD/HDD   │  μs-ms     TB
└───────────┘
```

## RAM (Random Access Memory)

**RAM** — оперативная память. Временное хранилище данных для работы процессора. При выключении данные теряются (volatile).

### Типы RAM

| Тип | Описание |
|-----|----------|
| **DRAM** | Требует обновления, дешевле — основная память |
| **SRAM** | Быстрее, дороже — используется в кэше CPU |

### Поколения DDR

| Тип | Частота | Пропускная способность |
|-----|---------|------------------------|
| DDR3 | 800-2133 MHz | 6.4-17 GB/s |
| DDR4 | 2133-3200 MHz | 17-25.6 GB/s |
| DDR5 | 4800-8400 MHz | 38.4-67.2 GB/s |

### Dual Channel
Две планки RAM = удвоенная пропускная способность (128-bit вместо 64-bit).

## ROM (Read-Only Memory)

**ROM** — постоянная память. Хранит данные без питания (non-volatile).

| Тип | Описание |
|-----|----------|
| **EEPROM** | Электрически стираемая |
| **Flash** | Современный тип (SSD, USB) |

Применение: BIOS/UEFI, прошивки устройств.

## Для Backend-разработчика

- **ECC RAM** — обязательна для серверов (исправляет ошибки)
- **Утечки памяти** — мониторить через `tracemalloc`, `memory_profiler`
- **In-memory DB** (Redis) — вся БД в RAM = молниеносный доступ
