## Ключевые концепции

- Вершины и рёбра, степень вершины
- Представления: матрица смежности O(V²), список смежности O(V+E)
- Обходы: BFS (поиск в ширину), DFS (поиск в глубину)
- Алгоритмы: Дейкстра, Беллман-Форд, Флойд-Уоршелл
- MST: алгоритмы Прима и Крускала
- Применение: карты, социальные сети, зависимости
