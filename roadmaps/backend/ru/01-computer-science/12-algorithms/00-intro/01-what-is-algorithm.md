## Определение

**Алгоритм** — это конечная последовательность чётко определённых инструкций, предназначенных для решения определённой задачи или выполнения вычисления.

Проще говоря, алгоритм — это "рецепт" для решения проблемы. Как кулинарный рецепт описывает шаги приготовления блюда, так и алгоритм описывает шаги решения задачи.

## Свойства алгоритма

Чтобы последовательность действий считалась алгоритмом, она должна обладать следующими свойствами:

### 1. Дискретность (пошаговость)
Алгоритм состоит из отдельных, чётко разделённых шагов.

### 2. Определённость (детерминированность)
Каждый шаг должен быть однозначно определён и не допускать различных толкований.

### 3. Конечность
Алгоритм должен завершаться за конечное число шагов.

### 4. Результативность
После выполнения алгоритма должен быть получен результат.

### 5. Массовость
Алгоритм применим к целому классу однотипных задач, а не только к одной конкретной.

### 6. Понятность
Алгоритм должен быть понятен исполнителю (человеку или компьютеру).

## Примеры из программирования

### Алгоритм нахождения максимального числа в списке

```python
def find_max(numbers):
    """
    Алгоритм поиска максимального элемента:
    1. Предположить, что первый элемент — максимальный
    2. Пройти по всем остальным элементам
    3. Если текущий элемент больше максимального — обновить максимум
    4. Вернуть найденный максимум
    """
    if not numbers:
        return None

    max_value = numbers[0]  # Шаг 1

    for num in numbers[1:]:  # Шаг 2
        if num > max_value:  # Шаг 3
            max_value = num

    return max_value  # Шаг 4


# Пример использования
numbers = [3, 7, 2, 9, 1, 5]
result = find_max(numbers)
print(f"Максимальное число: {result}")  # Максимальное число: 9
```

### Алгоритм проверки, является ли число чётным

```python
def is_even(number):
    """
    Алгоритм проверки чётности:
    1. Разделить число на 2
    2. Если остаток равен 0 — число чётное
    3. Иначе — число нечётное
    """
    if number % 2 == 0:
        return True
    else:
        return False


# Примеры
print(is_even(4))   # True
print(is_even(7))   # False
```

### Алгоритм линейного поиска

```python
def linear_search(arr, target):
    """
    Алгоритм линейного поиска:
    1. Пройти по каждому элементу массива
    2. Сравнить текущий элемент с искомым
    3. Если совпадает — вернуть индекс
    4. Если прошли весь массив и не нашли — вернуть -1
    """
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1


# Пример
data = ['apple', 'banana', 'cherry', 'date']
index = linear_search(data, 'cherry')
print(f"Элемент найден на позиции: {index}")  # Элемент найден на позиции: 2
```

## Типы алгоритмов

### По структуре выполнения:

1. **Линейные** — шаги выполняются последовательно, один за другим
2. **Разветвляющиеся** — содержат условия (if/else)
3. **Циклические** — содержат повторяющиеся действия (for, while)
4. **Рекурсивные** — вызывают сами себя

### По назначению:

1. **Поисковые** — находят элементы в структурах данных
2. **Сортировки** — упорядочивают элементы
3. **Графовые** — работают с графами и деревьями
4. **Криптографические** — шифруют и дешифруют данные
5. **Оптимизационные** — находят лучшее решение

## Способы записи алгоритмов

1. **Словесное описание** — на естественном языке
2. **Псевдокод** — формализованный язык, похожий на код
3. **Блок-схемы** — графическое представление
4. **Программный код** — на языке программирования

## Почему алгоритмы важны?

1. **Эффективность** — хороший алгоритм экономит время и ресурсы
2. **Масштабируемость** — правильный алгоритм работает с любым объёмом данных
3. **Предсказуемость** — алгоритм всегда даёт ожидаемый результат
4. **Основа программирования** — любая программа состоит из алгоритмов
