# Интерфейс GitHub

## Введение

Интерфейс GitHub может показаться перегруженным для новичка, но он логично организован и со временем становится интуитивным. В этом разделе разберём основные элементы интерфейса и научимся эффективно навигировать по платформе.

---

## 1. Навигация по сайту

### Верхняя панель навигации

На каждой странице GitHub присутствует верхняя панель:

```
┌─────────────────────────────────────────────────────────────────┐
│ 🐙 │ Search or jump to... │ PR │ Issues │ Marketplace │ Explore │ 🔔 │ ➕ │ 👤 │
└─────────────────────────────────────────────────────────────────┘
```

| Элемент | Описание |
|---------|----------|
| Логотип (Octocat) | Переход на главную страницу / Dashboard |
| Поиск | Поиск по всему GitHub или внутри репозитория |
| Pull Requests | Ваши открытые PR (созданные и назначенные) |
| Issues | Ваши Issues (созданные и назначенные) |
| Marketplace | Магазин приложений и Actions |
| Explore | Исследование популярных репозиториев |
| Уведомления (колокольчик) | Центр уведомлений |
| Плюс (+) | Быстрое создание репозитория, gist, организации |
| Аватар | Меню профиля и настройки |

### Клавиатурные сокращения

GitHub имеет множество hotkeys. Нажмите `?` на любой странице для просмотра:

```
Глобальные:
s или /    → фокус на поиске
g + n      → перейти к уведомлениям
g + d      → перейти к Dashboard
g + c      → перейти к Code (в репозитории)
g + i      → перейти к Issues
g + p      → перейти к Pull Requests

В репозитории:
t          → файловый поиск (fuzzy finder)
w          → переключение веток
l          → перейти к строке в файле
b          → blame view
.          → открыть VS Code в браузере (github.dev)
```

### Боковая панель (Sidebar)

В новом интерфейсе GitHub (2023+) появилась левая боковая панель:

```
┌────────────────────┐
│ 🏠 Home            │
│ 📋 Issues          │
│ 🔀 Pull Requests   │
│ 📁 Repositories    │
│ 💬 Discussions     │
│ 📚 Codespaces      │

├────────────────────┤
│ Recent repos:      │
│ - user/project1    │
│ - user/project2    │
└────────────────────┘
```

## 2. Dashboard (Главная страница)

После входа в аккаунт вы попадаете на Dashboard.

### Структура Dashboard

```
┌───────────────────────────────────────────────────────────────┐
│                     DASHBOARD                                  │
├─────────────────────────────┬─────────────────────────────────┤
│                             │  Repositories                    │
│  Recent Activity Feed       │  ┌─────────────────────────┐    │
│                             │  │ 🔍 Find a repository... │    │
│  • @user starred repo       │  ├─────────────────────────┤    │
│  • @friend pushed to repo   │  │ 📁 user/repo1           │    │
│  • New release: v2.0        │  │ 📁 user/repo2           │    │
│  • PR merged in project     │  │ 📁 org/project          │    │
│                             │  └─────────────────────────┘    │
│                             │                                  │
│                             │  Recent activity                 │
│                             │  Your teams                      │
│                             │                                  │
└─────────────────────────────┴─────────────────────────────────┘
```

### Секции Dashboard

**1. Activity Feed (Лента активности)**
- Действия людей, на которых вы подписаны
- Новости из репозиториев, которые вы watch
- Рекомендации на основе ваших интересов

**2. Repositories (Репозитории)**
- Быстрый доступ к вашим репозиториям
- Поиск по репозиториям
- Кнопка создания нового репозитория

**3. Recent Activity (Недавняя активность)**
- Ваши недавние действия
- Репозитории, которые вы посещали

**4. Explore (Исследование)**
- Trending репозитории
- Рекомендации по интересам
- Topics и Collections

### Настройка Dashboard

```
Settings → Appearance → Dashboard

Можно настроить:
- Показывать/скрывать секции
- Порядок отображения
- Тема (светлая/тёмная)
```

## 3. Feed (Лента новостей)

### Что отображается в Feed

```
┌─────────────────────────────────────────────────────────────────┐
│ ⭐ codemaster starred repository awesome-project                │
│    Python • Updated 2 hours ago                                 │
├─────────────────────────────────────────────────────────────────┤
│ 🔀 developer merged pull request in team/app                    │
│    "Add authentication module" #234                             │
├─────────────────────────────────────────────────────────────────┤
│ 🚀 project released v3.0.0                                      │
│    New features: OAuth, API v2, Performance improvements        │
├─────────────────────────────────────────────────────────────────┤
│ 🍴 friend forked popular-library                                │
│    from original-author/popular-library                         │
└─────────────────────────────────────────────────────────────────┘
```

### Типы событий

| Иконка | Событие | Описание |
|--------|---------|----------|
| ⭐ | Star | Кто-то добавил репозиторий в избранное |
| 🔀 | Merge | PR был слит |
| 🚀 | Release | Новый релиз |
| 🍴 | Fork | Репозиторий был форкнут |
| 👀 | Watch | Кто-то начал следить за репозиторием |
| 📝 | Push | Новые коммиты |

### Управление Feed

```
Подписаться на пользователя:
1. Перейти на профиль пользователя
2. Нажать "Follow"

Подписаться на репозиторий:
1. Перейти в репозиторий
2. Нажать "Watch" → выбрать уровень

Отписаться:
1. Те же действия, но "Unfollow" / "Unwatch"
```

## 4. Поиск репозиториев

### Строка поиска

```
┌─────────────────────────────────────────────────────────────────┐
│  🔍 Search or jump to...                                 [/]    │
└─────────────────────────────────────────────────────────────────┘
```

Нажмите `/` или `s` для быстрого перехода к поиску.

### Базовый поиск

```
Простой поиск:
react                    → репозитории с "react" в названии/описании
facebook/react           → конкретный репозиторий
@facebook                → репозитории пользователя/организации
```

### Расширенный поиск (Search Qualifiers)

```
По языку:
language:python flask

По количеству звёзд:
stars:>1000              → более 1000 звёзд
stars:100..500           → от 100 до 500 звёзд

По дате обновления:
pushed:>2024-01-01       → обновлялся после 1 января 2024

По размеру:
size:>1000               → больше 1 МБ

По лицензии:
license:mit              → с MIT лицензией

По типу:
is:public                → только публичные
is:private               → только приватные (ваши)

Комбинации:
language:javascript stars:>500 pushed:>2024-01-01 is:public
```

### Типы поиска

```
Repositories    → репозитории (по умолчанию)
Code            → код внутри репозиториев
Commits         → коммиты
Issues          → задачи и баги
Pull Requests   → пулл-реквесты
Discussions     → обсуждения
Users           → пользователи
```

### Поиск внутри репозитория

Находясь в репозитории:
```
1. Нажать 't' → fuzzy finder для файлов
2. Нажать '/' → поиск по коду в репозитории
3. Использовать вкладку "Code" → Search
```

### Сохранённые поиски

```
После поиска можно сохранить запрос:
1. Выполнить поиск
2. Нажать "Save" рядом с результатами
3. Дать название сохранённому поиску
4. Доступ через Dashboard или закладки
```

## 5. Notifications (Уведомления)

### Центр уведомлений

Нажмите на колокольчик 🔔 в верхней панели:

```
┌─────────────────────────────────────────────────────────────────┐
│ Notifications                                    Mark all read  │
├─────────────────────────────────────────────────────────────────┤
│ 📌 Inbox                                                        │
├─────────────────────────────────────────────────────────────────┤
│ 🔵 [project/repo] Issue: Bug in login #42                       │
│    @user mentioned you 2 hours ago                              │
├─────────────────────────────────────────────────────────────────┤
│ 🔵 [team/app] PR: Add feature #89                               │
│    Review requested 5 hours ago                                 │
├─────────────────────────────────────────────────────────────────┤
│ ⚪ [org/project] Release v2.1.0 published                       │
│    1 day ago                                                    │
└─────────────────────────────────────────────────────────────────┘
```

### Типы уведомлений

```
Participating (Участие):
- Вас упомянули (@username)
- Вы автор issue/PR
- Вам назначен review
- Вы подписаны на thread

Watching (Наблюдение):
- Любая активность в репозитории, который вы watch
- Новые issues, PR, releases
```

### Уровни Watch

Для каждого репозитория можно настроить:

```
Watch → Unwatch                 → никаких уведомлений
Watch → Participating only      → только когда вас упоминают
Watch → All activity            → все события
Watch → Ignore                  → полное игнорирование
Watch → Custom                  → выбрать конкретные события
```

### Действия с уведомлениями

```
┌──────────────────────────────────────────┐
│ Уведомление                              │
│                                          │
│ [Done] [Unsubscribe] [Save] [Mark read]  │
└──────────────────────────────────────────┘

Done        → архивировать
Unsubscribe → отписаться от thread
Save        → сохранить на потом
Mark read   → пометить прочитанным
```

### Фильтрация уведомлений

```
Filters:
reason:mention           → упоминания
reason:review-requested  → запросы на ревью
reason:assign            → назначения
is:unread               → непрочитанные
repo:owner/repo         → из конкретного репозитория
```

### Настройка уведомлений по email

```
Settings → Notifications

Можно настроить:
- Получать ли email для каждого уведомления
- Только digest (сводка)
- Отключить email полностью
- Разные настройки для разных типов событий
```

## 6. Страница репозитория

### Обзор структуры

```
┌─────────────────────────────────────────────────────────────────┐
│ owner / repository-name                          ⭐ Star  Fork   │
├───────────────────┬─────────────────────────────────────────────┤
│ <> Code          │ About                                        │
│ ◯ Issues      12 │ Description of the project                   │
│ ↗ Pull Req    3  │ 🌐 website.com                               │
│ 💬 Discussions   │ Topics: python, api, rest                    │
│ ▶ Actions        │ ────────────────────────                     │
│ 📊 Projects      │ ⭐ 1.2k  🍴 234  👁 45                         │
│ 📚 Wiki          │                                              │
│ 🔒 Security      │ Releases                                     │
│ 📈 Insights      │ v2.1.0 Latest                                │
│ ⚙ Settings       │                                              │
├───────────────────┼─────────────────────────────────────────────┤
│ Branch: main ▼   │ Add file ▼  | <> Code ▼  | ...              │
├───────────────────┴─────────────────────────────────────────────┤
│ 📁 src/                                                         │
│ 📁 tests/                                                       │
│ 📄 README.md                                                    │
│ 📄 package.json                                                 │
├─────────────────────────────────────────────────────────────────┤
│ README.md (rendered)                                            │
│ # Project Name                                                  │
│ Description...                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Вкладки репозитория

| Вкладка | Описание |
|---------|----------|
| Code | Файлы репозитория, README |
| Issues | Задачи, баги, обсуждения |
| Pull Requests | Запросы на слияние кода |
| Actions | CI/CD пайплайны |
| Projects | Канбан-доски для управления |
| Wiki | Документация |
| Security | Уязвимости, политики |
| Insights | Статистика, графики |
| Settings | Настройки (только для владельцев) |

### Кнопки действий

```
⭐ Star        → добавить в избранное (показывает поддержку)
👁 Watch      → подписаться на уведомления
🍴 Fork       → создать свою копию репозитория
<> Code       → скачать или клонировать
```

## 7. Просмотр кода

### Навигация по файлам

```
┌─────────────────────────────────────────────────────────────────┐
│ Branch: main ▼ │ Go to file │ Add file ▼ │ <> Code ▼          │
├─────────────────────────────────────────────────────────────────┤
│ 📁 src          │ Initial commit            │ 3 days ago       │
│ 📁 tests        │ Add unit tests            │ 2 days ago       │
│ 📄 README.md    │ Update documentation      │ 1 day ago        │
│ 📄 .gitignore   │ Initial commit            │ 3 days ago       │
└─────────────────────────────────────────────────────────────────┘
```

### Просмотр файла

При открытии файла:
- Синтаксическая подсветка
- Номера строк (кликабельные для ссылок)
- Кнопки: Raw, Blame, History, Edit, Delete

```
Горячие клавиши:
l → перейти к строке
b → blame view (кто менял каждую строку)
y → получить постоянную ссылку на эту версию
```

### Blame View

```
┌─────────────────────────────────────────────────────────────────┐
│ Blame for README.md                                             │
├─────────────────────────────────────────────────────────────────┤
│ abc123 @user  3 days │ 1 │ # Project Name                       │
│ abc123 @user  3 days │ 2 │                                      │
│ def456 @dev   1 day  │ 3 │ ## Installation                      │
│ def456 @dev   1 day  │ 4 │ Run `npm install`                    │
└─────────────────────────────────────────────────────────────────┘

Показывает:
- Хеш коммита
- Автор
- Когда изменено
- Содержимое строки
```

## 8. Настройки интерфейса

### Тема оформления

```
Settings → Appearance → Theme mode

Варианты:
- Light (светлая)
- Dark (тёмная)
- Dark dimmed (приглушённая тёмная)
- Sync with system (как в системе)
```

### Доступность

```
Settings → Accessibility

- Размер шрифта
- Анимации
- Клавиатурные сокращения
- Screen reader оптимизация
```

### Табы vs Пробелы

```
Settings → Appearance → Tab size preference

- Отображение табов как N пробелов
- По умолчанию: 8
```

## Заключение

Интерфейс GitHub насыщен функциональностью, но основные действия интуитивны:
- Используйте поиск (`/`) для быстрой навигации
- Настройте уведомления, чтобы не пропустить важное
- Изучите горячие клавиши (`?`) для ускорения работы
- Экспериментируйте с темами и настройками для комфорта

С опытом интерфейс станет привычным и удобным инструментом ежедневной работы.
