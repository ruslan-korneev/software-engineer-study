# Document Databases

Документоориентированные базы данных хранят данные в виде документов (обычно JSON/BSON).

## Что это?

Document DB — это тип NoSQL баз данных, где данные хранятся в виде полуструктурированных документов. Каждый документ содержит пары ключ-значение и может иметь вложенную структуру. Документы группируются в коллекции.

## Когда использовать

- Приложения с гибкой/изменяющейся схемой данных
- Хранение контента (CMS, блоги, каталоги)
- Профили пользователей с разными атрибутами
- Кэширование данных из внешних API
- Прототипирование и быстрая разработка

## Преимущества

- Гибкая схема данных
- Естественное представление объектов (JSON)
- Горизонтальное масштабирование
- Быстрые операции чтения/записи целых документов
- Поддержка вложенных структур
