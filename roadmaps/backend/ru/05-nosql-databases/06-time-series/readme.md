# Time Series Databases

Базы данных временных рядов оптимизированы для хранения и анализа данных с временной меткой.

## Что это?

Time Series DB — это специализированные базы данных для работы с данными, которые изменяются во времени. Каждая запись имеет timestamp, и данные обычно записываются последовательно, а читаются за временные диапазоны.

## Когда использовать

- Мониторинг инфраструктуры (метрики серверов, контейнеров)
- IoT и сенсорные данные
- Финансовые данные (котировки, транзакции)
- Application Performance Monitoring (APM)
- Логирование событий
- Аналитика в реальном времени

## Преимущества

- Оптимизированная запись (append-only)
- Эффективное сжатие временных данных
- Быстрые запросы по временным диапазонам
- Встроенные функции агрегации по времени
- Автоматическое удаление старых данных (retention)
