# Связные списки (Linked Lists)

Раздел посвящён связным спискам — структурам данных с динамическим размером и эффективными операциями вставки/удаления.

## Содержание

- [x] [Односвязный список](01-singly-linked.md) — узлы с указателем на следующий элемент
- [x] [Двусвязный список](02-doubly-linked.md) — узлы с указателями на предыдущий и следующий элементы
- [x] [Кольцевой список](03-circular-linked.md) — списки с замкнутым циклом

## Ключевые концепции

- Динамическое выделение памяти
- Вставка и удаление за O(1) при наличии указателя
- Отсутствие прямого доступа по индексу (O(n))
- Применение: реализация стеков, очередей, LRU-кэша
