# Криптовалюты

## Что такое криптовалюта

### Определение

**Криптовалюта** — это цифровая или виртуальная валюта, использующая криптографию для обеспечения безопасности транзакций. Криптовалюты работают на основе технологии блокчейн — распределённого реестра, который поддерживается сетью компьютеров (нод) по всему миру.

Ключевые характеристики криптовалют:
- **Децентрализация** — отсутствие центрального органа управления
- **Прозрачность** — все транзакции записываются в публичный блокчейн
- **Неизменяемость** — записанные транзакции нельзя изменить или удалить
- **Псевдоанонимность** — пользователи идентифицируются по адресам, а не по реальным именам

### История появления

Идея цифровых денег существовала задолго до Bitcoin:

- **1983** — Дэвид Чаум предложил концепцию электронных денег (eCash)
- **1998** — Wei Dai описал b-money, а Nick Szabo — bit gold
- **2008** — Сатоши Накамото опубликовал whitepaper "Bitcoin: A Peer-to-Peer Electronic Cash System"
- **2009** — Запуск сети Bitcoin, добыт первый блок (Genesis Block)
- **2015** — Запуск Ethereum, открывший эру смарт-контрактов

### Отличие от фиатных денег

| Характеристика | Криптовалюта | Фиатные деньги |
|----------------|--------------|----------------|
| Эмиссия | Алгоритмически ограничена | Контролируется центробанком |
| Управление | Децентрализованное | Централизованное |
| Прозрачность | Публичный блокчейн | Непрозрачная система |
| Границы | Глобальные, без ограничений | Национальные валюты |
| Посредники | Не требуются | Банки, платёжные системы |
| Инфляция | Предсказуемая или отсутствует | Зависит от политики ЦБ |

## Bitcoin (BTC)

### История создания

**Bitcoin** — первая и самая известная криптовалюта, созданная анонимным разработчиком (или группой) под псевдонимом **Satoshi Nakamoto**.

Ключевые даты:
- **31 октября 2008** — публикация whitepaper
- **3 января 2009** — создание Genesis Block с посланием: *"The Times 03/Jan/2009 Chancellor on brink of second bailout for banks"*
- **12 января 2009** — первая транзакция между Сатоши и Хэлом Финни
- **22 мая 2010** — "Bitcoin Pizza Day": 10 000 BTC за две пиццы
- **2011** — Сатоши исчезает из публичного пространства

### Основные характеристики

```
Алгоритм консенсуса: Proof of Work (SHA-256)
Время блока: ~10 минут
Награда за блок: 3.125 BTC (после халвинга 2024)
Максимальный размер блока: 1 MB (4 MB с SegWit)
Подтверждений для безопасности: 6 блоков (~1 час)
```

### Лимит 21 миллион

Bitcoin имеет **жёстко ограниченную эмиссию** — максимум 21 000 000 BTC. Это достигается через:

1. **Halvings (халвинги)** — уменьшение награды за блок вдвое каждые 210 000 блоков (~4 года)
   - 2009: 50 BTC
   - 2012: 25 BTC
   - 2016: 12.5 BTC
   - 2020: 6.25 BTC
   - 2024: 3.125 BTC

2. **Предсказуемая эмиссия** — последний Bitcoin будет добыт примерно в 2140 году

Ограниченная эмиссия делает Bitcoin **дефляционным активом**, часто сравниваемым с цифровым золотом.

### Роль в экосистеме

- **Store of Value (SoV)** — средство сохранения ценности
- **Расчётная единица** для крипторынка (торговые пары BTC/XXX)
- **Эталон** для сравнения других криптовалют
- **Резервный актив** для институциональных инвесторов

## Ethereum (ETH)

### Отличие от Bitcoin

**Ethereum** был создан Виталиком Бутериным в 2015 году как платформа для децентрализованных приложений.

| Аспект | Bitcoin | Ethereum |
|--------|---------|----------|
| Цель | Цифровые деньги | Платформа для dApps |
| Язык скриптов | Ограниченный | Тьюринг-полный (Solidity) |
| Время блока | ~10 минут | ~12 секунд |
| Эмиссия | 21 млн (лимит) | Нет жёсткого лимита |
| Консенсус | Proof of Work | Proof of Stake (с 2022) |

### Смарт-контракты

**Смарт-контракт** — это программа, выполняющаяся на блокчейне Ethereum при соблюдении определённых условий.

```solidity
// Простой пример смарт-контракта
pragma solidity ^0.8.0;

contract SimpleStorage {
    uint256 private storedValue;

    function set(uint256 value) public {
        storedValue = value;
    }

    function get() public view returns (uint256) {
        return storedValue;
    }
}
```

Применения смарт-контрактов:
- **DeFi** (децентрализованные финансы)
- **NFT** (невзаимозаменяемые токены)
- **DAO** (децентрализованные автономные организации)
- **Токенизация активов**

### Gas и комиссии

**Gas** — единица измерения вычислительных ресурсов в Ethereum.

```
Комиссия = Gas Used × Gas Price (Gwei)

Где:
- Gas Used — количество газа для операции
- Gas Price — цена за единицу газа
- 1 Gwei = 0.000000001 ETH
```

После обновления EIP-1559:
- **Base Fee** — базовая комиссия (сжигается)
- **Priority Fee (Tip)** — чаевые валидатору

### ETH 2.0 и переход на PoS

**The Merge** (сентябрь 2022) — переход Ethereum с Proof of Work на Proof of Stake:

**Преимущества PoS:**
- Снижение энергопотребления на ~99.95%
- Возможность стейкинга (минимум 32 ETH)
- Улучшенная масштабируемость (подготовка к шардингу)
- Дефляционная модель (сжигание части комиссий)

## Альткоины

### Определение

**Альткоины** (alternative coins) — все криптовалюты, кроме Bitcoin. Термин объединяет тысячи различных проектов с разными целями и технологиями.

### Категории альткоинов

#### 1. Платёжные криптовалюты
Предназначены для быстрых и дешёвых транзакций:
- **Litecoin (LTC)** — "серебро к золоту Bitcoin", время блока 2.5 минуты
- **Bitcoin Cash (BCH)** — форк Bitcoin с увеличенным размером блока
- **Dash** — приватные и мгновенные платежи

#### 2. Платформенные криптовалюты
Блокчейны для создания dApps:
- **Solana (SOL)** — высокая пропускная способность (65 000 TPS)
- **Cardano (ADA)** — академический подход, Proof of Stake
- **Polkadot (DOT)** — интероперабельность между блокчейнами
- **Avalanche (AVAX)** — быстрая финализация транзакций

#### 3. Утилитарные криптовалюты
Служат определённым целям в экосистемах:
- **Chainlink (LINK)** — оракулы для смарт-контрактов
- **Uniswap (UNI)** — governance токен DEX
- **Filecoin (FIL)** — децентрализованное хранение данных

### Примеры популярных альткоинов

| Монета | Год | Особенность |
|--------|-----|-------------|
| Litecoin | 2011 | Scrypt алгоритм, быстрые блоки |
| Ripple (XRP) | 2012 | Банковские переводы, централизация |
| Cardano | 2017 | Научный подход, Ouroboros PoS |
| Solana | 2020 | Proof of History, высокая скорость |
| Avalanche | 2020 | Субсети, быстрый консенсус |

## Стейблкоины

### Зачем нужны

**Стейблкоины** — криптовалюты, привязанные к стабильным активам (обычно к доллару США).

**Применения:**
- Защита от волатильности крипторынка
- Быстрые международные переводы
- Базовая валюта для DeFi протоколов
- Расчёты между трейдерами

### Типы стейблкоинов

#### 1. Обеспеченные фиатом
Каждый токен обеспечен реальными долларами в резерве:

- **USDT (Tether)** — крупнейший стейблкоин, спорная прозрачность резервов
- **USDC (Circle)** — регулируемый, аудированные резервы
- **BUSD (Binance USD)** — стейблкоин биржи Binance (прекращение выпуска в 2024)

#### 2. Обеспеченные криптовалютой
Обеспечены криптоактивами с избыточным залогом:

- **DAI (MakerDAO)** — децентрализованный, залог в ETH и других активах
- Коэффициент залога обычно 150%+

#### 3. Алгоритмические
Поддерживают привязку через алгоритмы без полного обеспечения:

- **FRAX** — частично обеспеченный
- **UST (Terra)** — крах в мае 2022 показал риски этой модели

### Риски стейблкоинов

1. **Контрагентный риск** — эмитент может не иметь полного обеспечения
2. **Регуляторный риск** — власти могут заморозить резервы
3. **Риск депега** — потеря привязки к доллару
4. **Централизация** — возможность блокировки адресов (USDT, USDC)

## Токены vs Монеты (Coins)

### Разница между token и coin

| Характеристика | Coin (Монета) | Token (Токен) |
|----------------|---------------|---------------|
| Блокчейн | Собственный | Работает на чужом блокчейне |
| Примеры | BTC, ETH, SOL | USDT, UNI, LINK |
| Создание | Требует разработки блокчейна | Деплой смарт-контракта |
| Применение | Оплата комиссий сети | Различные use-cases |

### ERC-20 токены

**ERC-20** — стандарт токенов на Ethereum, определяющий базовый интерфейс:

```solidity
interface IERC20 {
    function totalSupply() external view returns (uint256);
    function balanceOf(address account) external view returns (uint256);
    function transfer(address to, uint256 amount) external returns (bool);
    function allowance(address owner, address spender) external view returns (uint256);
    function approve(address spender, uint256 amount) external returns (bool);
    function transferFrom(address from, address to, uint256 amount) external returns (bool);
}
```

Другие стандарты токенов:
- **ERC-721** — NFT (невзаимозаменяемые токены)
- **ERC-1155** — мультитокены (fungible + non-fungible)
- **BEP-20** — аналог на BNB Chain

### Utility vs Security токены

#### Utility токены
Предоставляют доступ к продукту или услуге:
- Governance токены (UNI, AAVE)
- Токены доступа (BAT, FIL)
- Игровые токены (AXS, SAND)

#### Security токены
Представляют долю в активе или компании:
- Подпадают под регулирование ценных бумаг
- Требуют KYC/AML проверок
- Примеры: токенизированные акции, облигации

## Рыночные аспекты

### Market Cap (Рыночная капитализация)

```
Market Cap = Цена × Циркулирующее предложение

Пример:
BTC: $40,000 × 19,500,000 = $780 млрд
```

**Категории по капитализации:**
- **Large Cap** — >$10 млрд (BTC, ETH, BNB)
- **Mid Cap** — $1-10 млрд
- **Small Cap** — $100 млн - $1 млрд
- **Micro Cap** — <$100 млн

### Волатильность

Криптовалюты известны высокой волатильностью:

- Суточные колебания в 5-10% — норма
- Bitcoin в 2021: от $29,000 до $69,000
- Альткоины могут терять 90%+ от ATH

**Причины волатильности:**
- Небольшой размер рынка относительно традиционных активов
- Регуляторные новости
- Влияние крупных игроков ("киты")
- Спекулятивный характер рынка

### Биржи

#### Централизованные биржи (CEX)
- **Binance** — крупнейшая по объёму торгов
- **Coinbase** — публичная компания, регулируемая в США
- **Kraken** — известна безопасностью
- **OKX, Bybit** — популярны для деривативов

**Особенности CEX:**
- Удобный интерфейс
- Высокая ликвидность
- Требуют KYC
- Риск взлома/банкротства (Mt. Gox, FTX)

#### Децентрализованные биржи (DEX)
- **Uniswap** — крупнейший на Ethereum
- **PancakeSwap** — на BNB Chain
- **dYdX** — деривативы

**Особенности DEX:**
- Нет KYC
- Пользователь контролирует ключи
- Меньшая ликвидность
- Более высокие комиссии (gas)

## Заключение

Криптовалюты представляют собой новый класс цифровых активов, радикально меняющий представление о деньгах и финансовых системах. От Bitcoin как "цифрового золота" до Ethereum как платформы для децентрализованных приложений — экосистема продолжает развиваться, создавая новые возможности и вызовы.

Ключевые моменты для понимания:
- Bitcoin — первая и доминирующая криптовалюта с ограниченной эмиссией
- Ethereum — платформа для смарт-контрактов и dApps
- Альткоины решают различные задачи: платежи, платформы, утилиты
- Стейблкоины обеспечивают стабильность в волатильном рынке
- Понимание разницы между токенами и монетами важно для оценки проектов
