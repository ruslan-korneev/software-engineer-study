# Команда cd

## Что такое cd?

**cd** (Change Directory) — команда для перемещения между директориями в файловой системе. Это одна из самых часто используемых команд в терминале.

## Синтаксис

```bash
cd [директория]
```

## Базовое использование

### Переход в директорию
```bash
$ cd /home/user/documents
$ pwd
/home/user/documents
```

### Переход в домашнюю директорию
```bash
$ cd              # без аргументов — в домашнюю
$ cd ~            # то же самое
$ cd $HOME        # то же самое через переменную
```

### Переход в корневую директорию
```bash
$ cd /
$ pwd
/
```

## Специальные символы для навигации

### Текущая директория (.)
```bash
$ cd .            # остаться на месте (бесполезно для cd)
```

### Родительская директория (..)
```bash
$ pwd
/home/user/projects/app

$ cd ..           # на уровень вверх
$ pwd
/home/user/projects

$ cd ../..        # на два уровня вверх
$ pwd
/home/user
```

### Домашняя директория (~)
```bash
$ cd ~            # в свою домашнюю
$ cd ~/documents  # в documents внутри домашней
$ cd ~otheruser   # в домашнюю другого пользователя
```

### Предыдущая директория (-)
```bash
$ pwd
/home/user

$ cd /var/log
$ pwd
/var/log

$ cd -            # вернуться в предыдущую
/home/user

$ cd -            # и обратно
/var/log
```

## Примеры навигации

### Абсолютный путь (от корня)
```bash
$ cd /var/log
$ cd /home/user/projects
$ cd /etc/nginx/conf.d
```

### Относительный путь (от текущей директории)
```bash
$ pwd
/home/user

$ cd projects             # относительный путь
$ pwd
/home/user/projects

$ cd ../documents         # вверх и в другую папку
$ pwd
/home/user/documents
```

## Автодополнение с Tab

Tab — ваш лучший друг при навигации:

```bash
$ cd /ho<Tab>             # → /home/
$ cd /home/us<Tab>        # → /home/user/
$ cd ~/pro<Tab>           # → ~/projects/
```

Двойное нажатие Tab показывает варианты:
```bash
$ cd /etc/<Tab><Tab>
apache2/    nginx/    ssh/    systemd/    ...
```

## Работа с пробелами в именах

Если в имени директории есть пробелы:

```bash
# Способ 1: экранирование
$ cd /home/user/My\ Documents

# Способ 2: кавычки
$ cd "/home/user/My Documents"
$ cd '/home/user/My Documents'
```

## Переменные окружения

```bash
$ echo $PWD           # текущая директория
/home/user/projects

$ echo $OLDPWD        # предыдущая директория
/home/user

$ cd $OLDPWD          # альтернатива cd -
```

## Особенности cd

### cd — встроенная команда shell
В отличие от многих других команд, `cd` — это встроенная команда shell (builtin), а не отдельная программа:

```bash
$ type cd
cd is a shell builtin
```

### Ошибка при несуществующем пути
```bash
$ cd /nonexistent
bash: cd: /nonexistent: No such file or directory
```

### Права доступа
Для входа в директорию нужно право на исполнение (`x`):

```bash
$ cd /root
bash: cd: /root: Permission denied
```

## Полезные команды вместе с cd

### pushd и popd — стек директорий
```bash
$ pushd /var/log      # сохранить текущую и перейти
/var/log ~

$ pushd /etc          # ещё один переход
/etc /var/log ~

$ popd                # вернуться
/var/log ~

$ popd                # и ещё вернуться
~
```

### dirs — показать стек
```bash
$ dirs
/var/log /etc ~
```

## Советы

1. Используйте **Tab** для автодополнения — экономит время и уменьшает ошибки
2. `cd -` — быстро переключаться между двумя директориями
3. Настройте prompt чтобы видеть текущую директорию
4. `cd` без аргументов — быстрый возврат домой
