# Псевдокод

## Что такое псевдокод?

**Псевдокод** — это неформальный способ записи алгоритма, использующий структуру языка программирования, но записанный на естественном языке. Псевдокод не привязан к синтаксису конкретного языка программирования и служит для описания логики алгоритма.

## Зачем нужен псевдокод?

1. **Планирование** — помогает продумать алгоритм до написания кода
2. **Коммуникация** — понятен программистам, использующим разные языки
3. **Обучение** — легче понять логику без синтаксических деталей
4. **Документация** — описывает логику работы программы
5. **Отладка мышления** — выявляет логические ошибки на раннем этапе

## Основные элементы псевдокода

### 1. Переменные и присваивание

```
переменная = значение
имя = "Иван"
возраст = 25
сумма = a + b
```

### 2. Ввод и вывод

```
ВВОД: число
ВЫВОД: "Результат:", результат
ВЫВОД: сообщение
```

### 3. Условные конструкции

```
ЕСЛИ условие ТО
    действия
КОНЕЦ ЕСЛИ

ЕСЛИ условие ТО
    действия_если_истина
ИНАЧЕ
    действия_если_ложь
КОНЕЦ ЕСЛИ

ЕСЛИ условие1 ТО
    действия1
ИНАЧЕ ЕСЛИ условие2 ТО
    действия2
ИНАЧЕ
    действия3
КОНЕЦ ЕСЛИ
```

### 4. Циклы

```
ПОКА условие ДЕЛАТЬ
    действия
КОНЕЦ ПОКА

ДЛЯ i ОТ 1 ДО n ДЕЛАТЬ
    действия
КОНЕЦ ДЛЯ

ДЛЯ КАЖДОГО элемента В коллекции ДЕЛАТЬ
    действия
КОНЕЦ ДЛЯ

ПОВТОРЯТЬ
    действия
ПОКА условие
```

### 5. Функции

```
ФУНКЦИЯ имя_функции(параметры)
    тело функции
    ВЕРНУТЬ результат
КОНЕЦ ФУНКЦИИ

ВЫЗВАТЬ имя_функции(аргументы)
```

### 6. Массивы и списки

```
массив = [элемент1, элемент2, элемент3]
массив[индекс] = значение
элемент = массив[индекс]
ДЛИНА(массив)
```

### 7. Комментарии

```
// Это однострочный комментарий
/* Это многострочный
   комментарий */
```

## Примеры псевдокода

### Пример 1: Поиск максимального числа

```
ФУНКЦИЯ найти_максимум(числа)
    // Инициализация максимума первым элементом
    максимум = числа[0]

    // Проход по всем элементам
    ДЛЯ КАЖДОГО числа В числа ДЕЛАТЬ
        ЕСЛИ число > максимум ТО
            максимум = число
        КОНЕЦ ЕСЛИ
    КОНЕЦ ДЛЯ

    ВЕРНУТЬ максимум
КОНЕЦ ФУНКЦИИ
```

**Реализация на Python:**

```python
def find_max(numbers):
    maximum = numbers[0]

    for number in numbers:
        if number > maximum:
            maximum = number

    return maximum
```

### Пример 2: Проверка простого числа

```
ФУНКЦИЯ является_простым(n)
    // Числа меньше 2 не являются простыми
    ЕСЛИ n < 2 ТО
        ВЕРНУТЬ ЛОЖЬ
    КОНЕЦ ЕСЛИ

    // Проверяем делимость на числа от 2 до корня из n
    ДЛЯ i ОТ 2 ДО КОРЕНЬ(n) ДЕЛАТЬ
        ЕСЛИ n ДЕЛИТСЯ НА i ТО
            ВЕРНУТЬ ЛОЖЬ
        КОНЕЦ ЕСЛИ
    КОНЕЦ ДЛЯ

    ВЕРНУТЬ ИСТИНА
КОНЕЦ ФУНКЦИИ
```

**Реализация на Python:**

```python
import math

def is_prime(n):
    if n < 2:
        return False

    for i in range(2, int(math.sqrt(n)) + 1):
        if n % i == 0:
            return False

    return True
```

### Пример 3: Бинарный поиск

```
ФУНКЦИЯ бинарный_поиск(массив, искомое)
    левый = 0
    правый = ДЛИНА(массив) - 1

    ПОКА левый <= правый ДЕЛАТЬ
        середина = (левый + правый) / 2  // целочисленное деление

        ЕСЛИ массив[середина] == искомое ТО
            ВЕРНУТЬ середина
        ИНАЧЕ ЕСЛИ массив[середина] < искомое ТО
            левый = середина + 1
        ИНАЧЕ
            правый = середина - 1
        КОНЕЦ ЕСЛИ
    КОНЕЦ ПОКА

    ВЕРНУТЬ -1  // элемент не найден
КОНЕЦ ФУНКЦИИ
```

**Реализация на Python:**

```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1

    while left <= right:
        mid = (left + right) // 2

        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1
```

### Пример 4: Сортировка пузырьком

```
ФУНКЦИЯ сортировка_пузырьком(массив)
    n = ДЛИНА(массив)

    ДЛЯ i ОТ 0 ДО n-1 ДЕЛАТЬ
        // Флаг для оптимизации
        была_замена = ЛОЖЬ

        ДЛЯ j ОТ 0 ДО n-i-2 ДЕЛАТЬ
            ЕСЛИ массив[j] > массив[j+1] ТО
                // Обмен элементов
                временная = массив[j]
                массив[j] = массив[j+1]
                массив[j+1] = временная
                была_замена = ИСТИНА
            КОНЕЦ ЕСЛИ
        КОНЕЦ ДЛЯ

        // Если замен не было, массив отсортирован
        ЕСЛИ была_замена == ЛОЖЬ ТО
            ПРЕРВАТЬ
        КОНЕЦ ЕСЛИ
    КОНЕЦ ДЛЯ

    ВЕРНУТЬ массив
КОНЕЦ ФУНКЦИИ
```

**Реализация на Python:**

```python
def bubble_sort(arr):
    n = len(arr)

    for i in range(n - 1):
        swapped = False

        for j in range(n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True

        if not swapped:
            break

    return arr
```

### Пример 5: Факториал (рекурсивный)

```
ФУНКЦИЯ факториал(n)
    // Базовый случай
    ЕСЛИ n <= 1 ТО
        ВЕРНУТЬ 1
    КОНЕЦ ЕСЛИ

    // Рекурсивный случай
    ВЕРНУТЬ n * факториал(n - 1)
КОНЕЦ ФУНКЦИИ
```

**Реализация на Python:**

```python
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)
```

## Советы по написанию псевдокода

### DO (Делайте так):

1. **Используйте понятные имена** — `сумма`, `счётчик`, `максимум`
2. **Делайте отступы** — для обозначения вложенности
3. **Будьте последовательны** — используйте одни и те же ключевые слова
4. **Пишите кратко** — избегайте лишних деталей
5. **Добавляйте комментарии** — объясняйте сложные части

### DON'T (Не делайте так):

1. **Не используйте синтаксис конкретного языка** — `{}`, `;`, `def`
2. **Не углубляйтесь в детали реализации** — не нужны типы данных
3. **Не пишите слишком абстрактно** — должно быть понятно, что делает код
4. **Не смешивайте языки** — или на русском, или на английском

## Разные стили псевдокода

### Стиль 1: На русском языке (учебный)

```
ЕСЛИ возраст >= 18 ТО
    ВЫВОД: "Совершеннолетний"
ИНАЧЕ
    ВЫВОД: "Несовершеннолетний"
КОНЕЦ ЕСЛИ
```

### Стиль 2: На английском языке (международный)

```
IF age >= 18 THEN
    OUTPUT: "Adult"
ELSE
    OUTPUT: "Minor"
END IF
```

### Стиль 3: Близкий к коду

```
if (age >= 18)
    print("Adult")
else
    print("Minor")
```
