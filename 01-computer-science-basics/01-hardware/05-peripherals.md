# Peripherals

## Что такое периферия?

**Периферийные устройства** — внешние устройства для ввода, вывода и хранения данных.

## Классификация

### Устройства ввода
- Клавиатура, мышь
- Сканер, микрофон
- Веб-камера

### Устройства вывода
- Монитор, принтер
- Колонки, наушники

### Устройства ввода-вывода
- Внешние накопители (USB HDD/SSD)
- Сетевые карты
- Модемы

## Интерфейсы подключения

| Интерфейс | Скорость | Применение |
|-----------|----------|------------|
| USB 2.0 | 480 Mb/s | Клавиатура, мышь |
| USB 3.0 | 5 Gb/s | Внешние диски |
| USB-C 3.2 | 20 Gb/s | Современные устройства |
| Thunderbolt | 40 Gb/s | Профессиональное оборудование |
| HDMI 2.1 | 48 Gb/s | Мониторы 4K/8K |
| DisplayPort | 77 Gb/s | Мониторы |
| Ethernet | 1-100 Gb/s | Сеть |

## Для Backend-разработчика

### Серверная периферия

| Устройство | Назначение |
|------------|------------|
| **Сетевая карта (NIC)** | 10/25/100 Gb/s для серверов |
| **RAID-контроллер** | Управление дисковыми массивами |
| **GPU** | ML/AI вычисления |
| **HSM** | Аппаратное шифрование |

### Важные концепции

**Hot-swap** — замена устройства без выключения сервера (диски, БП)

**Redundancy** — дублирование критичных компонентов:
- 2+ БП
- 2+ сетевых карты
- RAID для дисков

**Out-of-band управление:**
- IPMI, iLO, iDRAC
- KVM over IP
- Доступ к серверу даже при зависшей ОС
