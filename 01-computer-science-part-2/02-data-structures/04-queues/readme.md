# Очереди (Queues)

Раздел посвящён очередям — структурам данных с принципом FIFO и их вариациям.

## Содержание

- [x] [Концепция очереди](01-queue-concept.md) — определение, операции, реализации
- [x] [Дек (Deque)](02-deque.md) — двусторонняя очередь
- [x] [Приоритетная очередь](03-priority-queue.md) — очередь с приоритетами, куча

## Ключевые концепции

- Принцип FIFO: первый вошёл — первый вышел
- Операции enqueue, dequeue, peek за O(1)
- Дек: операции с обоих концов за O(1)
- Приоритетная очередь: извлечение по приоритету, реализация на куче
- Применение: BFS, планировщики задач, буферизация
