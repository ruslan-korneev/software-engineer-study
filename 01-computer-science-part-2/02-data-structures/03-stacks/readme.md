# Стек (Stack)

Раздел посвящён стеку — структуре данных с принципом LIFO (Last In, First Out).

## Содержание

- [x] [Концепция стека](01-stack-concept.md) — определение, операции, реализации
- [x] [Применение стека](02-stack-applications.md) — практические примеры использования

## Ключевые концепции

- Принцип LIFO: последний вошёл — первый вышел
- Операции push, pop, peek за O(1)
- Применение: Call Stack, undo/redo, проверка скобок, DFS
- Реализация на массиве или связном списке
