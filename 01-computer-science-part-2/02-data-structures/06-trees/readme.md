## Ключевые концепции

- Иерархическая структура: корень, узлы, листья
- Обходы: preorder, inorder, postorder, level order
- BST: поиск, вставка, удаление за O(log n) в среднем
- Сбалансированные деревья: гарантия O(log n)
- Heap: извлечение min/max за O(log n)
- Trie: поиск по префиксу за O(m), где m — длина строки
