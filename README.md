## Как использовать

### Отслеживание прогресса
Каждая папка содержит `roadmap.md` с чекбоксами:
```markdown
- [x] Completed topic
- [ ] Not started yet
```

### Навигация
1. Открой главный `roadmap.md`
2. Перейди по ссылке на нужную тему
3. Изучи материал в `.md` файлах
4. Отметь `[x]` когда тема пройдена

### Заметки
Пустые `.md` файлы готовы для записи конспектов по мере изучения.
