## Two-file System

Each language folder uses two tracking files:

| File | Purpose | `[x]` means |
|------|---------|-------------|
| `roadmap.md` | Personal learning progress | Topic is studied and understood |
| `readme.md` | Content availability | File has content (>10 lines) |
