# Key-Value Databases

Key-Value (ключ-значение) — самый простой тип NoSQL баз данных.

## Что это?

Key-Value DB — это базы данных, которые хранят данные в виде пар «ключ → значение». По ключу можно получить, обновить или удалить значение. Значением может быть строка, число, JSON, бинарные данные и т.д.

## Когда использовать

- Кэширование (сессии, результаты запросов, страницы)
- Хранение сессий пользователей
- Rate limiting и счётчики
- Очереди сообщений
- Feature flags и конфигурация
- Leaderboards и рейтинги

## Преимущества

- Минимальная латентность (микросекунды)
- Простейшая модель данных
- Высокая производительность
- Отличное горизонтальное масштабирование
- In-memory хранение для критичных данных
