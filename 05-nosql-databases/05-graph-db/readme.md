# Graph Databases

Графовые базы данных хранят данные в виде узлов и связей между ними.

## Что это?

Graph DB — это базы данных, которые используют графовую модель данных: узлы (nodes), рёбра (edges) и свойства (properties). Они оптимизированы для работы с сильно связанными данными и выполнения запросов по связям.

## Когда использовать

- Социальные сети (друзья, подписки, рекомендации)
- Системы рекомендаций
- Fraud detection (обнаружение мошенничества)
- Knowledge graphs
- Управление зависимостями
- Сетевая инфраструктура

## Преимущества

- Быстрый обход связей (в отличие от JOIN в SQL)
- Естественное представление связанных данных
- Гибкая схема
- Мощный язык запросов (Cypher)
- Визуализация связей
