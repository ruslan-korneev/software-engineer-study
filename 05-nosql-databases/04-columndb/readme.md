# Column Databases

Колоночные базы данных оптимизированы для аналитических запросов над большими объёмами данных.

## Что это?

Column DB (колоночные/столбцовые БД) — это базы данных, которые хранят данные по столбцам, а не по строкам. Это позволяет эффективно читать только нужные колонки и сжимать данные с одинаковыми типами.

## Когда использовать

- OLAP и аналитика (отчёты, дашборды)
- Data Warehousing
- Логирование и мониторинг
- Обработка событий и clickstream
- Big Data обработка
- Business Intelligence

## Преимущества

- Высокая скорость аналитических запросов
- Эффективное сжатие данных (до 10x)
- Быстрая агрегация по колонкам
- Обработка петабайт данных
- Параллельная обработка запросов
